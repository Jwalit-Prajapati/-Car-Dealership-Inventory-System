# Inventory Management System

## Tech Stack
- Java 21/22
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- JUnit 5
- Mockito

## Features
- User Authentication
- Product Management
- Inventory Tracking
- Order Management

## Getting Started

### Run
./mvnw spring-boot:run

## Running Tests
./mvnw test

## Project Structure
src/
├── main/
└── test/

## Development Approach
This project follows Test-Driven Development (TDD):
1. Write a failing test (Red)
2. Implement the minimum code (Green)
3. Refactor while keeping tests passing
